This is the README for project piirustusohjelma.

Update: "piirustusohjelma.exe" is an application file that runs on Windows and does not need anything extra installed.

To be able to run the program you need:
    SIP 
        http://sourceforge.net/projects/pyqt/files/sip/sip-4.17/sip-4.17.zip
    PyQt5
        http://sourceforge.net/projects/pyqt/files/PyQt5/PyQt-5.5.1/PyQt5-5.5.1-gpl-Py3.4-Qt5.5.1-x32.exe
    Python 3.4
        can easily be found by googling
        
To run the program:
    1. go to src folder
    2. the file you want to run is main.py
